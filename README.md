# ForeverTogether 💍

A premium, single-page Wedding Invitation & RSVP Website template.

## Features
- 🎨 Elegant emerald & gold color palette
- ⏰ Live countdown timer
- 📖 Our Story timeline with alternating layout
- 🖼️ Masonry photo gallery
- 📅 Event details with Google Maps links
- 📝 RSVP form with Formspree integration
- 📱 Fully responsive with mobile hamburger menu
- ✨ AOS scroll animations throughout
- 💬 Romantic quote section
- 🎯 SEO optimized

## Quick Setup
1. Download `index.html`
2. Open in any text editor
3. Search for `[BUYER INSTRUCTION]` to find all customizable spots
4. Replace names, dates, images, and links
5. Upload to any web host

## RSVP Setup
1. Create a free account at [Formspree.io](https://formspree.io)
2. Create a new form
3. Copy your endpoint URL
4. Replace the `FORMSPREE_URL` placeholder in the code

## Tech Stack
- HTML5 + Tailwind CSS (CDN)
- Vanilla JavaScript
- AOS (Animate on Scroll)
- Google Fonts (Playfair Display, Inter, Cormorant Garamond)
